## 🎯 内容生成流程

```
选题生成 → 内容创作 → 内容优化 → 图片生成 → 内容审核
```

---

## 📝 选题生成

### 选题来源

1. **热点追踪**
   - 微博热搜
   - 知乎热榜
   - 小红书热门
   - 技术社区Trending

2. **关键词分析**
   - 用户搜索词
   - 平台推荐词
   - 竞品关键词

3. **用户需求**
   - 评论问题
   - 私信咨询
   - 社区讨论

4. **定期专题**
   - 每周技巧
   - 每月总结
   - 节日特辑

---

### 选题生成Prompt

```
你是一个专业的{领域}内容策划专家。

请根据以下信息生成5个选题：
- 热点话题：{hot_topics}
- 用户需求：{user_needs}
- 竞品分析：{competitor_analysis}

要求：
1. 标题吸引眼球
2. 内容有深度
3. 符合平台调性
4. 有差异化亮点

输出格式：
1. 标题 | 核心观点 | 目标用户
```

## ✍️ 内容创作

### 内容结构模板

```markdown
# 标题

【开场】
- 用故事/场景引入
- 引发共鸣

【核心内容】
### 要点1
- 具体内容
- 配图说明

### 要点2
- 具体内容
- 配图说明

### 要点3
- 具体内容
- 配图说明

【总结】
- 核心收获
- 行动建议

【互动】
- 提问引发讨论
- 引导收藏/分享

#标签
```

---

### 内容创作Prompt

```
你是一个专业的{领域}内容创作者。

请根据以下信息创作一篇小红书风格的图文内容：

选题：{topic}
核心观点：{core_point}
目标用户：{target_audience}

要求：
1. 标题：吸引眼球，包含关键词
2. 开场：用故事或场景引入
3. 内容：3-5个要点，每个要点200-300字
4. 结尾：总结+互动引导
5. 标签：5-10个相关标签
6. 风格：专业但不枯燥，有趣但不肤浅
7. 长度：800-1200字

输出：
- 标题
- 正文（Markdown格式）
- 标签
```

## ✨ 内容优化

### 标题优化

**优化原则**：
- 包含关键词
- 制造悬念
- 引发好奇
- 承诺价值

**优化Prompt**：
```
请优化以下标题，使其更吸引眼球：

原标题：{original_title}

要求：
1. 保留核心关键词
2. 增加吸引力
3. 符合小红书调性
4. 长度控制在20字以内

输出5个优化版本。
```

---

### 内容优化

**优化方向**：
- 增加可读性
- 优化排版
- 强化价值
- 增加互动

**优化Prompt**：
```
请优化以下内容：

{content}

优化方向：
1. 增加小标题
2. 优化段落结构
3. 增加实例说明
4. 强化互动引导
5. 精简冗余内容

保持原有风格和核心观点。
```

## 🔍 内容审核

### 审核维度

1. **内容质量**
   - 是否原创
   - 是否有价值
   - 是否有深度

2. **合规性**
   - 是否敏感
   - 是否侵权
   - 是否违规

3. **平台适配**
   - 是否符合调性
   - 是否符合规范
   - 是否有标签

---

### 审核Prompt

```
请审核以下内容：

标题：{title}
正文：{content}

审核维度：
1. 内容质量（1-10分）
2. 合规性（通过/不通过）
3. 平台适配性（1-10分）
4. 改进建议

输出审核报告。
```
