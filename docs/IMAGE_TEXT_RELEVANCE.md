# 图文相关性检查机制

## 问题背景

**发现的问题**：
- 之前发送的配图与笔记内容几乎无关
- 例如：笔记讲"华人数学家获奖"，图片却是普通数学学习场景
- 图文相关度接近0分

**根本原因**：
1. 搜索关键词过于泛泛（只搜"数学"）
2. 没有提取笔记核心主题关键词
3. 缺乏图文相关性检查机制
4. 没有图文匹配标准

---

## 解决方案

### 1. 关键词提取策略

**提取笔记核心关键词**：
- **主体关键词**：华人数学家、王虹、唐云清、科学家
- **事件关键词**：突破奖、获奖、封神、荣耀
- **情感关键词**：骄傲、成就、突破、国际影响力
- **场景关键词**：颁奖典礼、学术研究、数学公式

**搜索关键词组合**：
- 主搜索：`科学家 获奖 奖杯`
- 辅助搜索：`学术成就 荣耀 金色`
- 场景搜索：`颁奖典礼 科学突破`
- 情感搜索：`成功 庆祝 团队`

---

### 2. 场景化配图策略

**按笔记内容段落配图**：

```
笔记结构 → 配图策略：

1. 封面/标题
   - 图片：科学家+奖杯+荣耀感
   - 关键词：award, scientist, glory

2. 事件介绍
   - 图片：颁奖典礼、奖杯特写
   - 关键词：award ceremony, trophy

3. 人物介绍
   - 图片：学术场景、研究者工作
   - 关键词：research, scientist, laboratory

4. 内容展开
   - 图片：专业领域相关场景
   - 关键词：数学、公式、科技

5. 总结/情感
   - 图片：成功、团队、未来
   - 关键词：success, team, future
```

---

### 3. 图文相关性检查流程

**发布前必检清单**：

```
□ 标题字数 ≤ 20字（含emoji）
□ 图1与标题核心主题相关度 ≥ 4分
□ 图2与正文主要内容相关度 ≥ 4分
□ 图3与主题情感氛围匹配度 ≥ 4分
□ 图文整体一致性 ≥ 4分
□ 视觉风格统一性 ≥ 4分

总分 ≥ 20分 才能发布
```

**注意**：
- emoji也算1个字
- 标题超出20字必须修改
- 字数检查是第一优先级

**评分标准**：

| 分数 | 描述 | 示例 |
|------|------|------|
| 5分 | 高度相关，完美匹配 | 笔记讲获奖，图片是奖杯+科学家 |
| 4分 | 相关，基本匹配 | 笔记讲科学家，图片是研究者工作场景 |
| 3分 | 有联系，但不够紧密 | 笔记讲数学，图片是普通学习场景 |
| 2分 | 相关性较弱 | 笔记讲科技，图片是普通办公场景 |
| 1分 | 几乎无关 | 笔记讲AI，图片是自然风景 |
| 0分 | 完全不相关 | 笔记讲数学获奖，图片是美食 |

---

### 4. 内部检查流程

**Step 1: 提取笔记核心要素**
```
笔记标题：刚刚！华人数学家封神，斩获数学界"奥斯卡"
核心主题：华人科学家、获奖、突破奖
情感基调：荣耀、骄傲、成就
场景要素：颁奖、学术、数学
```

**Step 2: 确定搜索关键词**
```
关键词组1：科学家 获奖 奖杯 荣耀
关键词组2：学术成就 研究 突破
关键词组3：华人 国际影响力
```

**Step 3: 选择图片并打分**
```
图片1：团队合作成功场景
- 与"科学家成就"相关度：4分
- 与"荣耀"情感匹配：5分
- 与"华人影响力"相关：4分
→ 平均分：4.3分 ✅ 通过

图片2：庆祝荣耀时刻
- 与"获奖"相关度：5分
- 与"封神"情感匹配：5分
- 与"荣耀"相关：5分
→ 平均分：5分 ✅ 通过

图片3：专业工作场景
- 与"科学家研究"相关度：5分
- 与"数学家"相关：4分
- 与"学术"相关：5分
→ 平均分：4.7分 ✅ 通过
```

**Step 4: 总体评估**
```
图文整体一致性：4.5分
视觉风格统一性：4分
总分：22.5/25 = 90% ✅ 通过
```

## 工作流程改进

### 原流程（有问题）
```
笔记完成 → 搜索"数学" → 随机下载图片 → 发送
```

### 新流程（改进后）
```
笔记完成
↓
提取核心关键词（主体+事件+情感+场景）
↓
组合搜索关键词
↓
下载候选图片（多张）
↓
图文相关性评分（每张≥4分）
↓
总体评估（总分≥20分）
↓
通过 → 发送图片
不通过 → 重新搜索图片
```

---

## 持久化规则

**写入MEMORY.md的规则**：

1. **关键词提取**：
   - 必须提取笔记的：主体、事件、情感、场景
   - 组合成搜索关键词

2. **图文相关性检查**：
   - 发布前必须评分
   - 每张图片≥4分
   - 总分≥20分

3. **检查流程**：
   - 提取核心要素 → 确定搜索词 → 选择图片 → 打分评估 → 发布

4. **质量标准**：
   - 图文相关度 ≥ 80%
   - 视觉风格统一
   - 情感氛围匹配
