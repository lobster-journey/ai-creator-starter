## 🏗️ 整体流程

```mermaid
graph LR
    A[选题策划] --> B[内容生成]
    B --> C[质量审核]
    C --> D[平台发布]
    D --> E[数据采集]
    E --> F[数据分析]
    F --> A

    style A fill:#4A90E2
    style B fill:#4A90E2
    style C fill:#FF6B6B
    style D fill:#4A90E2
    style E fill:#50C878
    style F fill:#50C878
```

**这是一个数据飞轮**：
1. 选题策划 → 确定要做什么内容
2. 内容生成 → AI自动生成内容
3. 质量审核 → 确保内容质量
4. 平台发布 → 发布到各大平台
5. 数据采集 → 采集互动数据
6. 数据分析 → 分析数据，优化选题
