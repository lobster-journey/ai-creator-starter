---

### 审核实现

```python
class ContentAuditor:
    def __init__(self):
        self.sensitive_words = self.load_sensitive_words()

    def audit_text(self, text):
        # 敏感词检测
        for word in self.sensitive_words:
            if word in text:
                return {
                    'passed': False,
                    'reason': f'包含敏感词：{word}'
                }

        return {'passed': True}

    def audit_image(self, image_url):
        # 调用图片审核API
        result = self.image_audit_api.check(image_url)
        return result
```

## 🎯 合规建议

### 内容创作

1. **坚持原创**
   - 不抄袭他人内容
   - 不直接翻译外文
   - 添加个人观点

2. **事实准确**
   - 核实信息真实性
   - 引用权威来源
   - 避免虚假宣传

3. **价值导向**
   - 传递正能量
   - 避免负面情绪
   - 尊重公序良俗

---

### 图片使用

1. **原创优先**
   - 使用自己拍摄的图片
   - 使用自己设计的图片
   - AI生成图片（需标注）

2. **版权合规**
   - 使用免费图库
   - 获得授权使用
   - 标注来源

3. **人物肖像**
   - 获得当事人同意
   - 避免侵犯肖像权
   - 模特需有授权
